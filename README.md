# 🤖 Chatbot NTU

Đây là project **Chatbot hỗ trợ sinh viên Trường Đại học Nha Trang (NTU)**, được xây dựng nhằm hỗ trợ trả lời các câu hỏi thường gặp như:
- Thông tin học vụ
- Lịch học / lịch thi
- Thủ tục sinh viên
- Các câu hỏi về nhà trường
- Và các chức năng hỗ trợ khác

---

## 🚀 Công nghệ sử dụng

- Python 🐍
- Rasa / NLP (hoặc framework bạn dùng)
- Flask (API backend)
- SQLite / MySQL
- JavaScript / HTML / CSS

---

## 📌 Chức năng chính

- 💬 Chat trả lời tự động cho sinh viên NTU
- 📚 Hỗ trợ giải đáp thắc mắc
- 📅 Xem lịch học / lịch thi
- 🔐 Đăng nhập, giao diện trực quan tương tự hệ thống Elearning NTU
- ⚙️ Xử lý intent bằng NLP (Rasa / PhoBERT / model AI)
